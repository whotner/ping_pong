# Пинг Понг
Увлекательная простая игра, созданная на Python с использованием библиотеки Pygame. Ваша задача не дать мячу пролететь за окно: управляйте ракеткой и отбивайте приближающиещийся мяч

# Особенности игры
1. Простые управления ракетками 1 игрок на w и s, а 2 игрок на ↑ и ↓

2. Система подсчёта скорости: за каждое отбивание мяча прибавляется скорость

3. Увеличивающаяся сложность: скорость мяча растет

4. Визуальное отображение текущей скорости

5. Простой и понятный интерфейс

# Как запустить
1. Убедитесь, что у вас установлен Python

2. Установите библиотеку Pygame: `pip install pygame`

3. Скачайте файлы

4. Запустите файл ping_pong.py

# Требования
1. Python

2. Библиотека PyGame
